# 🌦️ Dashboard de Análise Climática em Excel

Dashboard desenvolvido no Microsoft Excel utilizando Tabelas Dinâmicas, Gráficos Dinâmicos e Timeline para análise de indicadores meteorológicos.

O projeto tem como objetivo demonstrar habilidades em análise de dados, organização de informações e construção de dashboards interativos utilizando recursos nativos do Excel.

## Indicadores Analisados
O dashboard apresenta informações como:
- Temperatura média
- Temperatura máxima
- Temperatura mínima
- Tempo de chuva
- Umidade relativa do ar
- Comparação mensal entre os anos de 2019 e 2020

## Dashboard
- O painel foi desenvolvido buscando uma visualização simples, organizada e intuitiva, permitindo analisar rapidamente os principais indicadores meteorológicos.

## Possíveis Melhorias Futuras

- Dashboard responsivo para impressão
- Segmentação por estação do ano
- Comparação percentual entre períodos
- Automatização da atualização dos dados com Power Query
